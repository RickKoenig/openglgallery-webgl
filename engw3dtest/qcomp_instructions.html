Input shortcuts:

  • Change current display mode:
    • Key:
      • 'q'   Q State mode
      • 'a'   Matrix Accumulate mode
      • 'o'   Matrix Column mode
      • 'e'   Expo! mode

  • Change current execution state:
    • Key:
      • Left Arrow
      • Right Arrow
      • Page Up
      • Page Down
    • Mouse:
      • Left
      • Right
    • UI Slider:
      • Left
      • Right

  • Change cursor QGate:
    • Key:
      • Up Arrow
      • Down Arrow
    • Mouse Wheel:
      • Up
      • Down

  • Drop QGate:
    • Mouse:
      • Left Button

  • Pickup QGate:
    • Mouse:
      • Right Button

  • Set column linkbacks (like Grover)
    • Mouse:
      • Middle Button (Once to select right column, again to link to a left column)
